About Me

Hello! My name is Tabitha, and I've been learning web design since I was 13 years old. I was in the eighth grade when my computer science teacher started teaching us how to create basic html web pages using tables to hold the content. I immediately fell in love with this, and started using what he taught us in my MySpace layouts. When MySpace went quiet, that was the end of me using HTML on a daily basis, until I enrolled in the local Career Tech High School where I took a class called Digital Design. In this class, we learned graphic design, video and web design. In my class, I was the only student who had previous experience with HTML, so I quickly picked up what she taught us, and I excelled in the class.

After high school, I attended The School of Advertising Art in Kettering Ohio, where I learned more about graphic design, photography video and web design. Half way through my first year, the web design teacher was offered his dream job, and put in his resignation at the school. He was replaced with the adjunct teacher who taught the typography course. Unfortunately, this new teacher was not proficient in in web design, and the course was a repeat of my high school class. Disappointed, I began looking into new schools to attend, because although I was learning a lot about design in general, I wanted to focus more on the web, and I knew that I was not going to be able to do so at SAA. So thats when I enrolled at Full Sail, and I've honestly been in love with it ever since.

Component Four (Featured) : Responsive Absolute Posistion Navigation

What I hope to learn

I hope to learn how to create a navigation bar that is responsive, and also absolute in its position, meaning that it will always be seen on screen by the user. Learning how to create this will help me in the future create better one page websites because then the user can always click to where they wish to go, instead of scrolling up and down to find the navigation or the information that they were looking for.

How this relates to what I've already learned

This is another element that I feel has almost became a web design standard, because it can be commonly found o on several web pages. I already know how to create navigation bars using standard lists, but I feel that learning how to create something like this is another skill that clients will be interested in me having.

How I might build this

  1. 1. Create your menu list in HTML
  2. 2. In CSS, style list so that it is displayed as "inline" and give it other styles that you want
  3. 3. In CSS, make the navigation menus position fixed.
  4. 4. At the bottom of the CSS file, create your responsive librarys, and style navigation to click into the size you want when the screen reaches that size you want.
